# Event-Management-System
A system to enhance and shortened the management procedure of any kind of event 

# Intro:<br>
Event Management System, is a console-based C program designed to simplify the overall event management process. The system helps manage event categories, services, customers, billing, and customer reviews in an organized way.<br>
# System Features:<br>
 -Login System – Secure access for Admin and Customer using username and password authentication.<br>
 -Admin Dashboard – Centralized control panel for managing events, services, customers, bills, and reviews.<br>
 -Event Category Management – Add, view, update, and delete different event categories such as Wedding, Birthday, Corporate Event, etc.<br>
 -Service Management – Manage services (e.g., Catering, Decoration, Photography) with service ID, name, and price.<br>
 -Customer Management – Store and manage customer registration details and login information.<br>
 -Event & Service Selection – Customers can view available event categories and services and select according to their preference.<br>
 -Billing System – Generate customer bills including event cost, service cost, VAT, and total amount, and store bill records using file handling.<br>
 -Bill Viewing System – Customers can view their own bills generated by the admin.<br>
 -Customer Preference System – Customers can add personal preferences related to their events.<br>
 -Review & Feedback System – Customers can submit reviews, and admins can view all customer feedback for service improvement.<br>
 -File Handling Based Storage – All data (admin, customer, category, service, bill, review) are stored using files without using any database, making the system lightweight and portable.<br>
# Advantages:<br>
 -An automated system reduces manual work and human errors in event planning and management.<br>
 -User-friendly, menu-driven interface makes the system easy to operate.<br>
 -Efficient management of event categories, services, customers, and billing.<br>
 -Generates useful records such as bills, customer lists, and reviews for better decision-making.<br>
 -Simple, lightweight, and fast because it is developed using the C programming language.<br>
# Limitations:<br>
· No graphical user interface (CLI-based system only).<br>
· Limited security (basic username and password authentication).<br>
· No online database or cloud storage support.<br>
· Bills and costs are entered manually by the admin.<br>
· Single-user access at a time (not multi-user supported).<br>
